# mySTrom Switch Integration

Diese Integration ermöglicht die Steuerung und Überwachung von mySTrom WiFi-Schaltsteckdosen in Home Assistant.

## Funktionen

- Schalten von mySTrom WiFi-Steckdosen
- Leistungsmessung in Echtzeit
- Energieverbrauchserfassung
- Temperaturüberwachung
- Integration mit dem Home Assistant Energy Dashboard
- Lokale Steuerung ohne Cloud-Verbindung

## Konfiguration

Die Integration kann über die UI konfiguriert werden:

1. Gehe zu Einstellungen → Geräte & Dienste
2. Klicke auf "Integration hinzufügen"
3. Suche nach "mySTrom Switch"
4. Gebe die IP-Adresse der mySTrom-Steckdose ein

## Unterstützte Geräte

- mySTrom WiFi Switch (CH)
- mySTrom WiFi Switch EU

Weitere Informationen findest du in der [vollständigen Dokumentation](https://github.com/proBieri/HAmyStrom).
